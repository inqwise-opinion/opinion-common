= Opinion Common

Opinion Common packages the shared domain primitives, utility classes, and
infrastructure glue that power the Opinion platform. The module is intended to
be imported by other Opinion services so they can rely on consistent
configuration, error handling, and integration patterns.

== Requirements

* Java 21 (Temurin 21 or newer JDK).
* Maven Wrapper (`./mvnw`) included with this repository.

== Add as a Dependency

[source,xml]
----
<dependency>
    <groupId>com.inqwise.opinion</groupId>
    <artifactId>opinion-common</artifactId>
    <version>${revision}</version>
</dependency>
----

Replace `${revision}` with the released version you wish to consume.

== Development

[subs="+quotes"]
----
# Compile sources #
./mvnw clean compile

# Run the unit test suite #
./mvnw clean test

# Produce source and javadoc jars #
./mvnw clean package source:jar javadoc:jar
----
